# RUNE Wave

Use this skill for RUNE work that changes product direction, descriptor
families, macro surfaces, generator interfaces, or VTRACE evidence.

## Workflow

1. Read `README.md`, `PRODUCT_PLAN.md`, and `context/waves/PHASES.md`.
2. Read the active wave and target pulse.
3. Confirm the change keeps RUNE core product-neutral.
4. Update VTRACE artifacts when mission, requirements, design, verification, or
   evidence changes.
5. Run the pulse validation commands.

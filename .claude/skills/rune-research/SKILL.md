# RUNE Research

Use this skill before broad RUNE adoption, generator design, macro expansion, or
IDL/schema compatibility decisions.

## Research questions

- What existing Rust macro, reflection, schema, and IDL systems solve part of
  this problem?
- Which descriptor concepts are stable enough for RUNE core?
- Which concepts should remain adapter-specific?
- What generated output is easiest for AI systems to inspect and validate?

## Output

Record findings with citations in a repo-local research note only when the user
explicitly asks for that artifact or a wave requires it.

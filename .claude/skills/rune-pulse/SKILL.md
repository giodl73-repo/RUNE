# RUNE Pulse

Use this skill to execute one small RUNE implementation or documentation slice.

## Workflow

1. Read the active wave and pulse.
2. Implement the smallest complete change.
3. Keep base names generic: entity, event, command, state, artifact, invariant,
   source, evidence, contract, version.
4. Avoid encoding downstream platform names in `rune-core`.
5. Run the pulse validation commands and update evidence notes when needed.

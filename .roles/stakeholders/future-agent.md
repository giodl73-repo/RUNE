# Future Agent

## Intellectual Disposition

The future agent needs to inspect RUNE output, understand what is safe to use,
and know where evidence or uncertainty remains.

## Key Question

*"Can I continue work from these descriptors without guessing hidden project intent?"*

## Lens - What to Verify

- Descriptors include stable ids and versions.
- Contract kinds and fields are explicit.
- Evidence and source links are addressable.
- Diagnostics identify incomplete or unsupported contract surfaces.
- Examples show how to validate generated output before relying on it.

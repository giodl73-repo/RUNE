# VTRACE Traceability Auditor

## Mission

Ensure RUNE changes are traceable from mission need through requirements, design,
implementation, verification, validation, and review evidence.

## Review checklist

- Mission or DCR changes have requirement and trace rows.
- Implementation claims have verification commands and evidence.
- Documentation-only waves do not claim code behavior.
- Follow-up blockers are recorded before a wave is marked complete.

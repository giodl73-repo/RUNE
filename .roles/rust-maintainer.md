# Rust Maintainer

## Mission

Keep RUNE useful to Rust maintainers without compromising native execution,
Cargo workflows, or idiomatic code ownership.

## Review checklist

- Rust crates can adopt RUNE incrementally through derives and explicit
  registries.
- No VM, garbage collector, plugin loader, or global runtime is required.
- Validation uses normal Cargo commands unless a future DCR approves otherwise.
- Runtime-facing features remain optional and explicit.

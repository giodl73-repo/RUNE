# RUNE tutorials

Tutorials are progressive. Follow the adopter path in order if you are adding
RUNE to a Rust crate for the first time.

## Adopter path

| Lesson | Outcome |
|---|---|
| [1. First contract](adopter-path/01-first-contract.md) | Annotate one Rust type with stable RUNE metadata. |
| [2. Registry and collection](adopter-path/02-registry-and-collection.md) | Expose an explicit registry and retain a collection fixture. |
| [3. Evidence commands](adopter-path/03-evidence-commands.md) | Inspect, check, inventory, and bundle evidence. |
| [4. Profiles and adapters](adopter-path/04-profiles-and-adapters.md) | Emit documentation and review packet outputs. |
| [5. Discovery manifest](adopter-path/05-discovery-manifest.md) | Merge retained collections through reviewed manifest discovery. |

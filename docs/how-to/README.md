# RUNE how-to guides

How-to guides are task-oriented. Each guide assumes you already understand the
basic RUNE model and want to complete one workflow.

| Guide | Task |
|---|---|
| [Derive a first contract](derive-first-contract.md) | Add `#[derive(RuneContract)]` metadata to a Rust type. |
| [Register contracts](register-contracts.md) | Expose a deterministic crate-owned registry. |
| [Generate retained evidence](generate-retained-evidence.md) | Inspect, check, inventory, and emit evidence from a collection fixture. |
| [Use a discovery manifest](use-discovery-manifest.md) | Merge approved retained collections deterministically. |
| [Generate profile and adapter output](generate-profile-and-adapter-output.md) | Emit documentation packets and review packets. |
| [Validate a semantic registry](validate-semantic-registry.md) | Check and inspect retained semantic registry evidence. |
| [Validate a state graph](validate-state-graph.md) | Check retained state graph evidence against a semantic registry. |
| [Validate an evidence packet](validate-evidence-packet.md) | Check retained diagnostic, validation, trace, health, and audit packet evidence. |
| [Validate an agent protocol request](validate-agent-protocol.md) | Check retained read-first agent protocol request evidence. |
| [Validate a compatibility report](validate-compatibility.md) | Check retained compatibility negotiation report evidence. |

For ordered end-to-end validation, use the
[adopter evidence validation runbook](../runbooks/adopter-evidence-validation.md).
